AUEvent uses the "OzFest Sunshine & Coral" color palette (v2).

**Core 4:**
- Primary: `#FF6B6B` (Sunset Coral) — buttons, CTA, active nav
- Secondary: `#4ECDC4` (Reef Mint) — header, sidebar accents
- Background: `#FFFAF5` (Sandy Cream) — page background
- Text: `#2C3E50` (Deep Navy) — body text

**Accents:**
- Sun Yellow: `#FFE66D` — recommended/highlight badges
- Festival Lavender: `#A29BFE` — category points
- Peach Glow: `#FFAB76` — sub-accents
- Mint Fresh: `#55EFC4` — free badges, nature category
- Blossom Pink: `#FD79A8` — spring events

**UI Helpers:** `--text-muted: #8898AA`, `--border: #FFE0DB`, `--card-bg: #FFFFFF`

Full palette doc: `docs/COLOR-PALETTE.md`

**Why:** Coral is universally warm across cultures (Korean, Chinese, Indian, Western). Mint represents Great Barrier Reef / Australian coastline. Cream background reduces eye fatigue for event-heavy info pages.
