AUEvent is an Australian event aggregator web app. MVP target is 8 weeks.

**Stack**: Next.js 16.2.0 (App Router), TypeScript, Tailwind CSS, deployed on Vercel.

**Team**: SSO (Lead Engineer) and 안종태 (Engineer) — 2 engineers total.

**Current state** (2026-03-20): Hello World page only at `frontend/app/page.tsx`. No backend yet. Static/mock data for Phase 3.

**Why**: Real-world data to come from Australian open data portals.

**How to apply**: Suggest solutions that are right-sized for a 2-engineer team. Avoid over-engineering. Prefer static/mock data until backend is scoped. Always read `frontend/node_modules/next/dist/docs/` before writing Next.js code — this is version 16.2.0 with breaking changes from earlier versions.
