# AUEvent

Australian event aggregator — fetching and normalising live event data from Australian open data portals.

## Project Structure

```
auevent/
├── frontend/   # Next.js web application
└── backend/    # API server (coming soon)
```

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Frontend**: Next.js, TypeScript, Tailwind CSS
- **Deployment**: Vercel

## Goals

- MVP delivery in 8 weeks
- Real-world data integration from Australian open data portals
